# Introduction

Welcome to the **Docusaurus Test Site**.

This is a sandbox environment for testing Docusaurus features before deploying them to production. Use this space to experiment with layouts, plugins, content structures, and configurations.

## What is this for?

This site is used to:

- **Test layouts** — experiment with page structure and navigation
- **Try plugins** — validate Docusaurus plugins before enabling them in production
- **Draft content** — write and review documentation without affecting live sites
- **Validate configs** — test `docusaurus.config.js` changes safely

## Getting started

Browse the docs using the sidebar on the left, or check the [Getting Started](getting-started/overview) section.
